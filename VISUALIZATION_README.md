# ChatFlow Throughput Visualization

This Python script visualizes throughput data generated by the ChatFlow client-part2.

## Requirements

```bash
pip install matplotlib
```

## Usage

### Basic Usage (auto-generate PNG and display chart)
```bash
python visualize_throughput.py throughput_20260213_140011.txt
```

This will:
- Generate a PNG file named `throughput_20260213_140011_chart.png`
- Display the chart in a window

### Custom Output File
```bash
python visualize_throughput.py throughput_20260213_140011.txt my_chart.png
```

## Chart Features

The generated chart includes:
- **Time series plot**: Throughput (messages/second) over time
- **Average line**: Red dashed line showing average throughput
- **Statistics box**: Max, min, and average throughput values
- **Grid**: For easier reading of values

## Example

```bash
# Visualize the latest throughput data
python visualize_throughput.py results/throughput_20260213_140011.txt

# Save with custom name
python visualize_throughput.py results/throughput_20260213_140011.txt analysis/my_throughput.png
```

## Output

- **Chart image**: High-resolution PNG (300 DPI) suitable for reports
- **Console output**: Summary statistics and data point count
